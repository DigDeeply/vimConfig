# vimConfig
vim config

Installation
---------------------
1. CheckOut

>	git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/vundle    
>	git clone git@github.com:DigDeeply/vimConfig.git	~/vimConfig

2. Link vimrc file

>	ln -s ~/vimConfig/vimrc ~/.vimrc

3. Install plugins

>	vim +BundleInstall +qall
